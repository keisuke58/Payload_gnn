## 1. 全体像: シミュレーション → 実測の橋渡し

```
【現在（FEM ベース）】
  Abaqus FEM → 応力/変位/温度場 → GNN → 欠陥検出
                ↑ シミュレーション

【将来（実測ベース）】
  PZT センサ → ガイド波信号 → 特徴量抽出 → GNN → 欠陥検出
                ↑ 実測データ

【橋渡し: Sim-to-Real】
  FEM で事前学習した GNN を、DANN (Domain Adaptation) で実測ドメインに適応
```

## 2. ガイド波 (Lamb Wave) とは

### 2.1 基本原理

薄板構造中を伝播する弾性波。圧電素子 (PZT) で励起し、別の PZT で受信する。

```
          励起 PZT                      受信 PZT
            ┃                              ┃
   ─────────╋──────────────────────────────╋─────── CFRP スキン (1mm)
            ┃         Lamb 波 →→→         ┃
   ═════════╬══════════════════════════════╬═══════ Al ハニカムコア (38mm)
            ┃                              ┃
   ─────────╋──────────────────────────────╋─────── CFRP スキン (1mm)

   健全部: 波がスムーズに伝播
   欠陥部: 反射・散乱・振幅変化・位相遅れ が発生
```

### 2.2 主要モード

| モード | 特性 | 欠陥検出での役割 |
|--------|------|-----------------|
| **S0** (対称) | 高速 (~5000 m/s), 低分散 | 長距離伝播、面内欠陥の検出 |
| **A0** (反対称) | 面外変位大, 高分散 | **スキン-コア デボンディング検出に最適** |
| **Skin Mode** | スキン内に閉じ込め | 局所的な剥離検出（高周波帯） |

### 2.3 デボンディングで何が起こるか

```
健全部:                          剥離部:
  波速 ~2500 m/s                   波速 ~1500 m/s (低下)
  振幅 = 基準値                    振幅 = 増幅 (エネルギートラップ)
  位相 = 基準値                    位相 = 遅延
  境界条件: "supported"            境界条件: "free" (コアと分離)
```

スキン-コア界面が剥離すると、波の伝播特性が局所的に変化する。
この変化パターンが「欠陥のシグネチャ」として GNN の入力になる。

## 3. ハードウェア構成

### 3.1 計測システム全体

```
┌─────────────┐    ┌──────────────┐    ┌──────────────┐
│ 波形生成器   │───→│ リニアアンプ  │───→│ PZT 励起素子  │
│ (Function   │    │ (±70V 出力)  │    │ (送信)       │
│  Generator) │    └──────────────┘    └──────┬───────┘
└─────────────┘                               │
                                         構造体(フェアリング)
┌─────────────┐    ┌──────────────┐           │
│ PC / 解析   │←───│ DAQ          │←───┌──────┴───────┐
│ ソフトウェア │    │ (A/D変換     │    │ PZT 受信素子  │
│             │    │  10MHz, 14bit)│    │ (受信 ×複数) │
└─────────────┘    └──────────────┘    └──────────────┘
```

### 3.2 主要コンポーネント

| 機器 | 仕様 | 役割 |
|------|------|------|
| **PZT センサ** | PZT-5A, 直径 8-10mm, 厚さ 0.5-1mm | ガイド波の送受信 |
| **波形生成器** | 任意波形, ~MHz 帯域 | 励起信号の生成 |
| **リニアアンプ** | ±70V, ~500kHz 帯域 | PZT 駆動電圧の増幅 |
| **DAQ (データ収集)** | サンプリング 10MHz, 14-16bit | 受信信号のデジタル化 |
| **PC** | 信号処理 + ML 推論 | データ保存・解析・GNN 推論 |

| パラメータ | 値 |
|-----------|-----|
| **配置面** | フェアリング内壁（内スキン表面） |

| **配置パターン** | 格子状 or 放射状 |
| **合計センサ数** | 1/12セクター: ~30-50個、全周: ~400-600個 |
| **接着方法** | エポキシ接着 or 共硬化埋め込み |

## 4. 計測手順（1回の走査）

### Step 1: 励起信号の送信

```python
# 典型的な励起信号: 5サイクル Hann 窓付き正弦バースト
f_center = 100e3    # 中心周波数 100 kHz
n_cycles = 5        # サイクル数
amplitude = 70      # ±70V

# t = [0, ..., n_cycles/f_center]
signal = amplitude * sin(2*pi*f_center*t) * hann_window(t)
```

送信波形（5サイクル トーンバースト）:
```
振幅
 70V ┤     ╱╲   ╱╲
     │   ╱    ╲╱    ╲
  0V ┤──╱──────────────╲──────────
     │ ╱                 ╲
-70V ┤╱                    ╲╱
     └──────────────────────────→ 時間 (~50μs)
```

### Step 2: Pitch-catch 全経路走査

```
センサ配列: S1, S2, S3, ... Sn

Round 1: S1 送信 → S2,S3,...Sn 受信  (n-1 信号)
Round 2: S2 送信 → S1,S3,...Sn 受信  (n-1 信号)
  ...
Round n: Sn 送信 → S1,S2,...Sn-1 受信

合計: n × (n-1) 個の時系列信号
```
