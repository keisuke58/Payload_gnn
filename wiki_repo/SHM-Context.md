# SHM コンテキスト: 構造ヘルスモニタリングの技術背景

> 最終更新: 2026-02-28

## 1. 目的と対象

| 項目 | 内容 |
| :--- | :--- |
| **対象構造** | CFRP スキン / Al-Honeycomb コア サンドイッチ構造 |
| **対象欠陥** | スキン-コア界面のデボンディング (剥離) |
| **駆動力** | CTE 不整合熱応力 + 音響疲労 (147–148 dB) + 衝撃 + 製造欠陥 |
| **対象機体** | H3 フェアリング (φ5.2m) + PSS |
| **課題** | φ5.2m × 10–16m の巨大表面積 → 手動 NDI は高コスト → **自動 SHM** の価値大 |

## 2. ガイド波 SHM の物理

### 2.1 Lamb 波モード

| モード | 特性 | 用途 |
| :--- | :--- | :--- |
| **S₀** (対称) | 高速, 低分散, 面内変位 | 長距離伝播, 面内欠陥 |
| **A₀** (反対称) | 面外変位大, 高分散 | **デボンディング検出に最適** |
| **Skin Mode** | 高周波, スキン内に閉じ込め | 局所的なスキン-コア剥離検出 |

### 2.2 サンドイッチ構造での伝播特性

- **Leaky Waves**: 波動エネルギーがスキンからコアへ漏洩し、反対側スキンへ到達
- **高減衰**: ハニカムコア + 接着層がダンパとして機能 → 高周波 (>100 kHz) は急速に減衰
- **Global Mode**: サンドイッチ全体の曲げ波 (低周波)
- **Skin Mode**: スキン内に閉じ込められたレイリー波的モード (高周波) — **SHM に最適**

### 2.3 デボンディングの波動シグネチャ

剥離が存在すると、局所的な境界条件が変化:

```
健全部: Skin → [bonded] → Core → [bonded] → Skin
  境界条件: "supported" (拘束あり)
  → 波速: 高 (サンドイッチとして振る舞い)

剥離部: Skin → [free] → Core
  境界条件: "free" (拘束なし)
  → 波速: 低 (薄板として振る舞い)
```

- **トラップエネルギー**: 波動が剥離領域にトラップされ、振幅増幅・リンギング (定在波) が発生
- **位相速度低下**: サンドイッチ → 薄板への遷移で位相速度が低下
- **散乱パターン**: A₀ モードの散乱が GNN の入力特徴量となる

## 3. CFRP 固有の課題

### 3.1 異方性分散

- 波速が伝播方向と繊維配向の相対角度に依存
- [45/0/-45/90]s 準等方性積層でも完全な等方性ではない
- → GNN が方向依存の散乱パターンを学習する必要

### 3.2 高減衰

- ハニカムコアが高周波を吸収
- センサ間隔 **30–50 cm** が必要
- 周波数選択: **50–300 kHz** (Skin Mode 帯域)

### 3.3 温度感度

- CTE 不整合 (CFRP: −0.3×10⁻⁶ vs Al: 23×10⁻⁶ /°C)
- 温度変化で見かけの欠陥信号が発生
- → **温度補償** が必須 (本プロジェクトでは温度を GNN 特徴量に含める)

### 3.4 曲率効果

- 円筒形状でモード変換・集束/発散が発生
- 平板モデルの直接適用不可
- → **曲率対応グラフ構築** (法線・主曲率を特徴量に含める)

## 4. JAXA 規格要件

### JAXA-JMR-002 (Payload Safety Standard)

- 安全重要複合材構造に **"No-Growth"** 要件
  - 検査不能な欠陥は、供用荷重下で成長しないことを実証する必要
  - SHM は定期 NDI の代替として **"Condition-Based Maintenance" (CBM)** を可能にする
- 安全率: SF ≥ 1.4 (降伏) / 1.5 (破壊)

### F8 事故後の意義

- F8 で PSS の製造時剥離が飛行中に進展し破壊に至った
- "No-Growth" が満たされていなかった
- → SHM による製造品質検査 + 運用中モニタリングの両方が必要

## 5. GNN アプローチの根拠

### なぜ CNN ではなく GNN か

| 課題 | CNN の限界 | GNN の利点 |
| :--- | :--- | :--- |
| **曲面構造** | 2D投影で歪み発生 | 非ユークリッド多様体上で直接動作 |
| **不規則メッシュ** | 正規グリッド前提 | 任意トポロジーのグラフに対応 |
| **局所性** | 固定カーネルサイズ | k-NN/測地線距離で適応的近傍 |
| **物理整合性** | ブラックボックス | メッセージパッシング ≈ FEM剛性行列演算 |
| **スケーラビリティ** | 解像度に依存 | ノード数に対して線形 |

### ノード特徴量設計 (18次元)

| # | 特徴量 | 次元 | 物理的意味 |
| :---: | :--- | :---: | :--- |
| 1–3 | 座標 $(x, y, z)$ | 3 | 空間位置 |
| 4–6 | 法線 $(n_x, n_y, n_z)$ | 3 | 局所曲率のエンコード |
| 7–8 | 主曲率 $(\kappa_1, \kappa_2)$ | 2 | ガウス/平均曲率 |
| 9–11 | 応力 $(\sigma_{xx}, \sigma_{yy}, \tau_{xy})$ | 3 | FEM 計算応力 |
| 12 | 温度 $T$ | 1 | 熱環境 |
| 13–15 | 熱応力 | 3 | CTE 不整合由来 |
| 16 | ノードタイプ | 1 | スキン/コア/欠陥 |
| 17–18 | 測地線距離特徴 | 2 | グラフ上の距離情報 |

## 7. 関連技術

### Acoustic Emission (AE)

- パッシブ監視: 亀裂進展時の弾性波を受信
- 制限: 打ち上げ中の音響環境 (~148 dB) でノイズに埋もれる
- 用途: 静的試験・地上検査で有効

### Fiber Bragg Grating (FBG)

- EMI免疫, 軽量, 多重化可能
- JAXA の極低温タンク (H3 LH2) でのひずみ監視に研究中
- フェアリングへの適用も将来的に検討可能
