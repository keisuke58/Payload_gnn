## 1. 目的と対象

| 項目 | 内容 |
| :--- | :--- |
| **対象構造** | CFRP スキン / Al-Honeycomb コア サンドイッチ構造 |
| **対象欠陥** | スキン-コア界面のデボンディング (剥離) |
| **駆動力** | CTE 不整合熱応力 + 音響疲労 (147–148 dB) + 衝撃 + 製造欠陥 |
| **対象機体** | H3 フェアリング (φ5.2m) + PSS |
| **課題** | φ5.2m × 10–16m の巨大表面積 → 手動 NDI は高コスト → **自動 SHM** の価値大 |

### 1.1 位置づけと従来技術との比較

本研究は、従来の非破壊検査 (NDI) とは異なり、構造ヘルスモニタリング (SHM) のデジタルツイン側を担うものです。

**非破壊評価 (NDE) の体系:**
```
非破壊評価 (NDE)
├── NDI/NDT — 従来の定期検査（超音波、X線、打音...）
└── SHM — 構造ヘルスモニタリング（常時監視）
    ├── センサベース（PZTアレイ + ガイド波）  ← 将来の実機実装 (Year 2~)
    └── デジタルツイン（FEM + ML）            ← 本研究の現段階 (Year 1)
```

**従来NDI vs 本研究 (GNN-SHM) の比較:**

| 項目 | 従来 NDI | 本研究 (GNN-SHM) |
| :--- | :--- | :--- |
| **タイミング** | 定期点検時（オフライン） | 常時監視（オンライン/リアルタイム） |
| **手法** | 超音波探傷、X線、打音検査 | FEMシミュレーション + GNN推論 |
| **人手** | 熟練検査員が必要 | 自動化・自律化 |
| **対象** | 検査可能な箇所のみ（局所） | 構造全面（グローバル） |
| **目的** | 欠陥の「有無」判定 | 欠陥の 位置・サイズ・種別 まで特定 |

本プロジェクトは現在、FEMで生成した応力・変位・温度場からGNNが欠陥を検出する**デジタルツイン**のフェーズにあります。将来的にはセンサ実測データ（ガイド波など）に接続し、ドメイン適応 (DANN) による Sim-to-Real を経て、**リアルタイムNDI** として機能させる構想です（2年計画の Year 2）。

## 2. 使い捨てフェアリングに対する SHM の意義（なぜこの研究か）

H3 フェアリングは**再利用しない**（分離後は回収・再使用しない）。「使い捨てなのに SHM の意味はあるのか？」という問いに対する答え。

### 2.1 打上中の事故防止

フェアリングは1回限りでも、**打上中に破壊されれば衛星もロケットも失う**。

- **F8 事故** (2025年): PSS の接着不良が破壊の一因。デボンディングが Max Q 付近で進展し破壊に至った。これは従来の定期NDIでは見逃される可能性のある欠陥が、実際の飛行環境下で致命的になることを示しており、**SHMによる常時監視**の重要性を裏付けている。
- 打上前に欠陥を検出できれば、**その1回の打上を確実に成功**させられる
- 使い捨てだからこそ、「その1回」の成否が全て

### 2.2 打上前の品質管理

- 製造時のボイド・脱層、輸送・組立中の損傷を**打上前**に検出
- SHM 手法（センサ配置、GNN による欠陥検出）は**地上での非破壊検査**にも転用可能
- 製造品質のスクリーニングとして価値がある

### 2.3 技術の汎用性

| 応用先 | 内容 |
| :--- | :--- |
| **将来の再利用ロケット** | 日本でも再利用機の研究が進む。手法はそのまま転用可能 |
| **航空機・宇宙機** | CFRP サンドイッチは航空機にも多用。手法は汎用 |
| **GNN-SHM 一般** | グラフ + FEM + ML の組み合わせは他構造にも応用可能 |

### 2.4 学術・産業的価値

- **GNN による SHM**: 未開拓テーマ。論文・特許の余地
- **CFRP/Al 界面のデボンディング**: 材料・構造の基礎研究として重要
- **JAXA 就職**: このテーマは宇宙輸送技術部門・航空技術部門と直結

**結論**: 使い捨てだからこそ、その1回の打上を確実に成功させることが重要。打上前の欠陥検出・品質管理、および将来の再利用機・他構造への技術移転という観点で、本研究には明確な意義がある。

## 3. ガイド波 SHM の物理

### 3.1 Lamb 波モード

| モード | 特性 | 用途 |
| :--- | :--- | :--- |
| **S₀** (対称) | 高速, 低分散, 面内変位 | 長距離伝播, 面内欠陥 |
| **A₀** (反対称) | 面外変位大, 高分散 | **デボンディング検出に最適** |
| **Skin Mode** | 高周波, スキン内に閉じ込め | 局所的なスキン-コア剥離検出 |

### 2.2 サンドイッチ構造での伝播特性

- **Leaky Waves**: 波動エネルギーがスキンからコアへ漏洩し、反対側スキンへ到達
- **高減衰**: ハニカムコア + 接着層がダンパとして機能 → 高周波 (>100 kHz) は急速に減衰
- **Global Mode**: サンドイッチ全体の曲げ波 (低周波)
- **Skin Mode**: スキン内に閉じ込められたレイリー波的モード (高周波) — **SHM に最適**

### 2.3 デボンディングの波動シグネチャ

剥離が存在すると、局所的な境界条件が変化:

```
健全部: Skin → [bonded] → Core → [bonded] → Skin
  境界条件: "supported" (拘束あり)
  → 波速: 高 (サンドイッチとして振る舞い)

剥離部: Skin → [free] → Core
  境界条件: "free" (拘束なし)
  → 波速: 低 (薄板として振る舞い)
```

- **トラップエネルギー**: 波動が剥離領域にトラップされ、振幅増幅・リンギング (定在波) が発生
- **位相速度低下**: サンドイッチ → 薄板への遷移で位相速度が低下
- **散乱パターン**: A₀ モードの散乱が GNN の入力特徴量となる

## 4. CFRP 固有の課題

### 4.1 異方性分散

- 波速が伝播方向と繊維配向の相対角度に依存
- [45/0/-45/90]s 準等方性積層でも完全な等方性ではない
- → GNN が方向依存の散乱パターンを学習する必要

### 3.2 高減衰

- ハニカムコアが高周波を吸収
- センサ間隔 **30–50 cm** が必要
- 周波数選択: **50–300 kHz** (Skin Mode 帯域)

### 4.3 温度感度

- CTE 不整合 (CFRP: −0.3×10⁻⁶ vs Al: 23×10⁻⁶ /°C)
- 温度変化で見かけの欠陥信号が発生
- → **温度補償** が必須 (本プロジェクトでは温度を GNN 特徴量に含める)

### 3.4 曲率効果

- 円筒形状でモード変換・集束/発散が発生
- 平板モデルの直接適用不可
- → **曲率対応グラフ構築** (法線・主曲率を特徴量に含める)

## 5. JAXA 規格要件

### JAXA-JMR-002 (Payload Safety Standard)

- 安全重要複合材構造に **"No-Growth"** 要件
  - 検査不能な欠陥は、供用荷重下で成長しないことを実証する必要
  - SHM は定期 NDI の代替として **"Condition-Based Maintenance" (CBM)** を可能にする
- 安全率: SF ≥ 1.4 (降伏) / 1.5 (破壊)

### F8 事故後の意義

- F8 で PSS の製造時剥離が飛行中に進展し破壊に至った
- "No-Growth" が満たされていなかった
- **SHMによる常時監視**があれば、飛行中（または発射直前）に異常な剥離進展を検知できた可能性がある。
- → SHM による製造品質検査 + 運用中モニタリングの両方が必要

## 5. GNN アプローチの根拠

### なぜ CNN ではなく GNN か

| 課題 | CNN の限界 | GNN の利点 |
| :--- | :--- | :--- |
| **曲面構造** | 2D投影で歪み発生 | 非ユークリッド多様体上で直接動作 |
| **不規則メッシュ** | 正規グリッド前提 | 任意トポロジーのグラフに対応 |
| **局所性** | 固定カーネルサイズ | k-NN/測地線距離で適応的近傍 |
| **物理整合性** | ブラックボックス | メッセージパッシング ≈ FEM剛性行列演算 |
| **スケーラビリティ** | 解像度に依存 | ノード数に対して線形 |

## 10. FAQ: センサ実装と運用

### Q1: センサ実測データ（特徴量）は具体的にどうやって取得するのか？

FEMで使用している「応力・変位・温度場」などの特徴量は、実機では以下のようなセンサ技術を用いて取得・再構成します。

1.  **変位・振動（ガイド波）**:
    *   **PZT（圧電素子）**: 構造表面に貼り付けたPZT素子がアクチュエータ（波の送信）とセンサ（波の受信）の両方として機能します。
    *   ガイド波（Lamb波）を伝播させ、その散乱波をPZTアレイで受信することで、構造全体の健全性を評価します。
2.  **歪み・応力**:
    *   **FBG（光ファイバーセンサ）**: 軽量で電磁ノイズに強く、フェアリングのような大型複合材構造に適しています。製造時に埋め込むことで、内部の歪み分布を計測可能です。
    *   **ひずみゲージ**: 局所的な応力集中箇所のモニタリングに使用します。
3.  **温度**:
    *   熱電対やFBGセンサを用いて取得します。温度変化はガイド波の速度に影響を与えるため、補正データとして重要です。

デジタルツインのアプローチでは、これら「点」や「線」で得られたセンサデータを、カルマンフィルタやデータ同化技術を用いてFEMモデルに反映させ、構造全体の「場」の状態を推定してからGNNに入力します。あるいは、GNN自体が疎なセンサデータから直接損傷を推定するように学習されます（Year 2の課題）。

### Q2: ロケット打ち上げ中にもリアルタイムで取得するのか？

**はい、その通りです。**

*   **リアルタイム監視の目的**:
    *   打ち上げ時の激しい振動や空力荷重によって、製造時の微細な欠陥が進展しないか（No-Growthの確認）、あるいは新たな損傷が発生していないかを監視します。
*   **データ伝送と処理**:
    *   ロケットから地上へのテレメトリ帯域は限られているため、全センサの生波形データを送信するのは困難です。
    *   そのため、**オンボード（機上）でのエッジ処理**が想定されます。軽量化されたGNNモデルが機上で推論を行い、「異常の有無」や「損傷レベル」といった低容量の結果データのみを地上へ送信します。
    *   詳細なデータは、オンボードメモリに記録し、回収可能な実験機や、あるいは異常発生時のみ優先的に送信するなどの運用が考えられます。
