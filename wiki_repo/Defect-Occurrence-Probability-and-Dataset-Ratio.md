# 欠陥発生確率とデータセット割合 — Defect Occurrence Probability & Dataset Ratio

> 最終更新: 2026-02-28  
> 文献調査に基づく欠陥発生確率と、データセット作成時の推奨割合をまとめる。

---

## 1. 概要

GNN-SHM 用データセットを設計する際、**欠陥タイプ別のサンプル割合**と**健全 vs 欠陥の比率**をどう設定すべきかは、学習性能と実運用への適合性に直結する。本ページでは文献・産業実態に基づく推奨値を示す。

---

## 2. 文献調査：欠陥発生の実態

### 2.1 航空宇宙複合材の欠陥発生要因

| 欠陥タイプ | 発生要因 | 文献・根拠 |
|------------|----------|------------|
| **製造時接着不良 (debonding)** | OoA 接着の不完全固化、異物混入 | F8 事故で PSS 4個全てに剥離確認。製造検査が主因。NASA NTRS 20160005994 |
| **内スキン-コア (inner_debond)** | 同上 | 外スキンと同様の製造プロセス。発生頻度は外より低い傾向 |
| **異物混入 (FOD)** | 製造工程での FOD 混入 | MDPI Appl. Sci. 2024。ハニカムセル内への金属片・工具片 |
| **衝撃損傷 (impact)** | 取扱い、異物衝突、BVID | ASTM D7136。在役中に発生。製造後検査で検出されることも |
| **積層内層間剥離 (delamination)** | マトリクス割れの進展、製造時の空隙 | Compos. Sci. Technol. 2006。多層積層で発生しやすい |
| **熱応力進展 (thermal_progression)** | CTE 不整合の繰返し熱サイクル | Composites Part B 2018。在役経年で進展 |
| **音響疲労 (acoustic_fatigue)** | 打ち上げ 147–148 dB | UTIAS 2019。衝撃損傷があると寿命短縮 |

### 2.2 発生確率の推定（文献ベース）

**注意**: 航空宇宙分野では欠陥発生率の公表データが限られる。以下は研究・産業報告から推定した相対的な目安。

| 欠陥タイプ | 相対発生頻度（推定） | 根拠 |
|------------|----------------------|------|
| **debonding** | **高 (25–40%)** | F8 事故で製造時接着不良が有力。フェアリング PSS で最も懸念される |
| **fod** | 中 (10–20%) | 製造工程の FOD 管理に依存。完全排除は困難 |
| **impact** | 中 (10–20%) | 在役中の取扱い・衝突。BVID は目視検出限界付近 |
| **delamination** | 中 (10–15%) | 積層構造に固有。マトリクス割れから進展 |
| **inner_debond** | 低〜中 (5–15%) | 外スキンより荷重条件が穏やかだが、製造要因は同様 |
| **thermal_progression** | 低 (5–10%) | 経年・熱サイクル蓄積。初期欠陥が存在する場合に進展 |
| **acoustic_fatigue** | 低 (5–10%) | 打ち上げ環境固有。衝撃損傷と相互作用 |

### 2.3 健全 vs 欠陥の実運用比率

| 分野 | 健全:欠陥（サンプルレベル） | 根拠 |
|------|-----------------------------|------|
| **航空機・宇宙構造の定期検査** | 95:5 〜 99:1 | 異常は稀。品質管理が効いている |
| **製造検査（出荷前）** | 90:10 〜 95:5 | 不良品はスクリーニングで排除 |
| **研究・ベンチマーク用** | 50:50 〜 80:20 | 学習の安定性と検出性能評価のバランス |

**docs/NOISE_AUGMENTATION_AND_CLASS_RATIO.md** より:
- 実運用に近い評価: **90:10 〜 95:5**
- 学習の安定性: **70:30 〜 80:20**
- バランス重視（検出性能上限）: **50:50**

## 3. データセット作成の推奨割合

### 3.1 欠陥タイプ別サンプル割合（推奨）

文献・F8 事故の示唆を踏まえた推奨値:

| 欠陥タイプ | 推奨割合 | 理由 |
|------------|----------|------|
| **debonding** | **25–30%** | 最優先。外スキン-コアが F8 と同様の破壊モード |
| **fod** | 12–15% | 製造工程で発生しうる |
| **impact** | 12–15% | BVID は NDT 検出の重要対象 |
| **delamination** | 12–15% | 積層構造に固有 |
| **inner_debond** | 8–10% | 内スキン-コア。発生頻度は外より低め |
| **thermal_progression** | 8–10% | 経年進展。在役 SHM で重要 |
| **acoustic_fatigue** | 8–10% | 打ち上げ環境。衝撃との相互作用 |

**現行 generate_doe.py のデフォルト** (`TYPE_FRACTIONS`):
```
debonding 25%, fod 15%, impact 15%, delamination 15%,
inner_debond 10%, thermal_progression 10%, acoustic_fatigue 10%
```
→ 上記推奨と整合的。

### 3.2 健全サンプル vs 欠陥サンプルの比率

| 目的 | 健全:欠陥 | 実現方法 |
|------|------------|----------|
| **Phase 1（パイプライン検証）** | 1:50 〜 1:100 | 健全 1 + 欠陥 50–100。学習は不均衡だが検証には十分 |
| **Phase 2（ベンチマーク学習）** | 1:1 〜 4:1 | 健全をノイズ拡張で水増し。50:50 または 80:20 |
| **Phase 3（実運用寄り評価）** | 9:1 〜 19:1 | 健全 90–95%、欠陥 5–10%。複数健全 FEM または大規模ノイズ拡張 |

**実装例** (docs/NOISE_AUGMENTATION_AND_CLASS_RATIO.md):
- **50:50**: 健全 1 → ノイズ拡張 99 バリアント → 100:100
- **80:20**: 健全 1 → 400 バリアント、欠陥 100 → 400:100

### 3.3 サンプル数の目安

| フェーズ | 欠陥サンプル | 健全サンプル | 合計 | 用途 |
|----------|--------------|--------------|------|------|
| Phase 1 | 50 | 1 | 51 | パイプライン検証 |
| Phase 2 | 200–300 | 50–200 | 250–500 | モデル性能評価 |
| Phase 3 | 500+ | 500–2000 | 1000–2500 | 本格学習・交差検証 |
