# H3 ロケット総覧

> 最終更新: 2026-02-28

## 1. 概要

| 項目 | 値 |
| :--- | :--- |
| **開発** | JAXA + 三菱重工業 (MHI) |
| **目的** | H-IIA/B 後継の次世代基幹ロケット（今後30年運用） |
| **設計思想** | "使いやすいロケット" — モジュラー構成・低コスト・高信頼性 |
| **全長** | 約 57 m (構成による) |
| **初飛行** | 2023年3月 (TF1 — 失敗), 2024年2月 (TF2 — 初成功) |
| **LEO能力** | 最大 ~6.5 t (H3-24 構成) |
| **GTO能力** | 最大 ~6.5 t (H3-24 構成) |

## 2. 構成命名規則: H3-**abc**

| 桁 | 意味 | 選択肢 |
| :--- | :--- | :--- |
| **a** (エンジン数) | 第1段 LE-9 基数 | 2 or 3 |
| **b** (ブースター数) | SRB-3 本数 | 0, 2, or 4 |
| **c** (フェアリング) | フェアリング型式 | **S** (Short), **L** (Long), **W** (Wide) |

例:
- **H3-22S** = LE-9×2, SRB-3×2, Short fairing — 最も多い運用構成
- **H3-24W** = LE-9×2, SRB-3×4, Wide fairing — 大型ペイロード用 (HTV-X)
- **H3-30S** = LE-9×3, SRB-3×0, Short fairing — 日本初ブースターなし大型液体ロケット

## 3. エンジン仕様

### 第1段: LE-9

| 項目 | 値 |
| :--- | :--- |
| **サイクル** | エキスパンダーブリードサイクル (世界初の第1段用) |
| **推力** | ~1,471 kN (真空) / ~1,220 kN (海面) |
| **推進剤** | LH2 / LOX |
| **比推力** | ~425 s (真空) |
| **基数** | 2基 or 3基 |
| **特徴** | LE-7A (二段燃焼) より構造が簡素 → 信頼性向上・コスト低減 |
| **製造** | MHI |

### 第2段: LE-5B-3

| 項目 | 値 |
| :--- | :--- |
| **サイクル** | エキスパンダーブリードサイクル |
| **推力** | ~137 kN (真空) |
| **推進剤** | LH2 / LOX |
| **燃焼時間** | 最大 **740 s** (LE-5B-2 の 534 s から大幅延長) |
| **改良点** | LH2ターボポンプ耐久性向上, ミキサー改良による燃費改善 |
| **基数** | 1基 |
| **製造** | MHI |

> LE-5B-3 は TF1 および F8 の両事故で問題が発生した系統。信頼性向上が最優先課題。

### 固体ロケットブースター: SRB-3

| 項目 | 値 |
| :--- | :--- |
| **推力** | ~2,158 kN |
| **ケース材** | CFRP (フィラメントワインディング) |
| **共通性** | Epsilon S の第1段と共通 |
| **本数** | 0, 2, or 4本 |
| **製造** | IHI エアロスペース |

- 2敗とも第2段系統の問題 (TF1: LE-5B-3 着火失敗, F8: PSS 破壊)

## 5. 現状 (2026年2月)

- **F8 事故調査** が進行中 (→ [[F8-Accident-Analysis]])
  - 衛星搭載構造 (PSS) 内部の CFRP/Al-HC 界面剥離が有力要因
  - 製造済み PSS 4個全てで想定を超える剥離を確認
- F9 以降の打ち上げは調査完了まで**凍結**
- 国産ロケットによる打ち上げが一時的に停止状態
- 基幹ロケット有識者検討会が定期的に進捗を報告

## 6. 本プロジェクトとの関連

H3 フェアリング (CFRP/Al-HC サンドイッチ) の構造健全性モニタリングが本プロジェクトの直接の研究対象。F8 事故により、この構造形式の接着健全性が「理論上の課題」から「現実の喫緊課題」に格上げされた。
