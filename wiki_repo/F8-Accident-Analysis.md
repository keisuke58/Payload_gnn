# F8 事故分析: H3ロケット8号機 PSS 破壊

> 最終更新: 2026-02-28 (JAXA 2026/02/25 調査報告に基づく)

## 1. 事故概要

| 項目 | 内容 |
| :--- | :--- |
| **日時** | 2025年12月22日 10:51:30 JST |
| **構成** | H3-22S |
| **ペイロード** | QZS-5 (みちびき5号) |
| **射場** | 種子島宇宙センター |
| **結果** | **失敗** — 衛星の軌道投入不能 |

第1段飛行は正常に完了。第2段エンジン LE-5B-3 の第1回燃焼は正常だったが、**フェアリング分離直後に衛星搭載構造 (PSS) が破壊**し、衛星が第2段から脱落。その後の第2回燃焼も異常停止し、ミッション失敗。

## 2. PSS (Payload Support Structure) とは

衛星搭載構造 (PSS) は、ロケット第2段と衛星を接続する**円筒形アダプタ**。

| パラメータ | 仕様 |
| :--- | :--- |
| **構造** | CFRP スキン / Al-Honeycomb コア サンドイッチ |
| **構成** | **4分割パネル** を接着接合 |
| **接合方式** | H3では常圧加温接着 (OoA方式) |
| **機能** | 衛星荷重の支持・伝達、分離機構のベース |

> PSS の構造形式 (CFRP/Al-HC サンドイッチ) は**フェアリングと同一**。フェアリング製造技術がそのままPSSにも適用されている。

## 3. 破壊メカニズム (調査中)

```
[打ち上げ前]
  PSS 内部に製造時の CFRP/Al-HC 界面剥離が存在
  (検査済み PSS 4個全てで想定を超える剥離を確認)
    ↓
[上昇中 — 高度上昇に伴う外部真空化]
  パネル内部の残留空気 (~1 atm) と外部真空の圧力差
    → 既存の剥離が段階的に進展
    ↓
[フェアリング分離 (火工品点火)]
  分離衝撃がトリガー
    → 62 ms 以内に PSS 構造が座屈・破壊
    → 衛星が第1段側に落下
    → 第2段の LH2 タンク加圧系統にも損傷
    → 第2回燃焼が早期停止
```

## 4. 調査の進捗 (2026年2月時点)

### FTA (故障の木解析)

- 当初 6項目 → **5項目** に絞り込み
- フェアリング分離時の直接接触は排除
- 残る有力要因:
  - **製造時の接着不良 (デボンディング)** ← 最有力
  - 圧力差による剥離進展
  - 分離衝撃による座屈

### 製造検査の結果

- 検査済み PSS **4個全て** で想定を超える剥離を確認
- CFRP スキンと Al-Honeycomb コアの界面に空隙
- 製造工程 (常圧加温接着) に起因する可能性

## 5. 本研究への示唆

> **F8 事故は、CFRP/Al-Honeycomb サンドイッチ構造の接着健全性モニタリングが「理論的な研究テーマ」ではなく「現実の喫緊課題」であることを実証した。**

| 事故の教訓 | 本プロジェクトでの対応 |
| :--- | :--- |
| 製造時の接着不良が見逃された | → GNN による欠陥位置特定で検出精度向上 |
| 飛行中に剥離が進展した | → CTE 不整合による熱応力を FEM で再現し GNN に学習 |
| 目視検査では検出困難 | → Lamb波ガイド波 SHM による非接触・大面積検査 |
| PSS もサンドイッチ構造 | → フェアリング以外の構造にも応用可能 |
| 4個中4個に剥離 | → 製造プロセス全体の品質モニタリングが必要 |

### 研究の正当性強化

F8 以前: 「将来的にSHMが必要になる**かもしれない**」
F8 以後: 「SHMが**現に必要**であることが実証された」

- JAXA-JMR-002 の "No-Growth" 要件が CFRP 接着構造に直接適用
- PSS の製造品質問題は、同一製造技術を用いるフェアリングにも潜在的に影響
- 本プロジェクトの GNN-SHM は、この課題に対する直接的な解決策を提供
