## 装置メーカー3社（詳細）

### 1. 東京エレクトロン (TEL)

| 項目 | 内容 |
|------|------|
| 主力製品 | エッチング装置、コータ/デベロッパ、成膜装置(CVD/ALD)、洗浄装置 |
| コア物理 | プラズマ物理、反応化学、気体分子運動論 |
| シミュレーション | PIC-MCC, DSMC, 反応CFD, プロセスシミュレーション |
| 商用ソフト | COMSOL Multiphysics, ANSYS Fluent/CFX |
| 言語 | Python, C++, Fortran |
| 平均年収 | ~1,400万円 |
| 勤務地 | 山梨(韮崎), 宮城, 岩手, 熊本 |

**ポートフォリオ**: `portfolio/tel/`
- `pic_mcc_plasma.py` — 2D PIC-MCC Ar プラズマシミュレータ
- `etch_profile_levelset.py` — Level-Set エッチングプロファイル

**内定戦略**:
- プラズマPICコードの自作が最も刺さる
- エッチングプロファイル予測の実装経験
- COMSOL でのマルチフィジックス解析経験

### 2. ディスコ (DISCO)

| 項目 | 内容 |
|------|------|
| 主力製品 | ダイシングソー(世界シェア70-80%), グラインダ, ポリッシャ |
| コア物理 | 材料力学, 振動工学, 伝熱工学, 破壊力学 |
| シミュレーション | FEM構造/熱, モーダル解析, CFD(クーラント), 接触解析 |
| 商用ソフト | ANSYS Mechanical/Fluent, **Abaqus**, NASTRAN, LS-DYNA |
| 言語 | Python, MATLAB |
| 平均年収 | **~1,672万円**（業界最高水準） |
| 勤務地 | 東京(大田区), 広島(呉) |

**ポートフォリオ**: `portfolio/disco/`
- `grinding_fem.py` — FEM 熱-構造連成 研削シミュレータ
- `vibration_modal.py` — 超音波ダイシングブレード振動モード解析

**内定戦略**:
- **Abaqus/FEM経験が直結**（現研究と最も親和性が高い）
- 研削シミュレーション + AI品質予測のデモ
- 精密加工プロセスの力学モデル理解

### 3. レーザーテック (Lasertec)

| 項目 | 内容 |
|------|------|
| 主力製品 | EUVマスクブランクス検査装置(世界唯一), DUVマスク検査, SiCウエハ検査 |
| コア物理 | 光学(波動/幾何), 電磁気学, レーザー物理 |
| シミュレーション | FDTD, RCWA, BPM, 光線追跡, 画像処理+AI |
| 商用ソフト | Ansys Lumerical FDTD, Zemax OpticStudio, CODE V |
| 言語 | Python, C++, MATLAB |
| 平均年収 | ~1,638万円 |
| 勤務地 | 横浜(港北区) |

**ポートフォリオ**: `portfolio/lasertec/`
- `fdtd_2d.py` — 2D FDTD 電磁波ソルバ (DUV 193nm マスク欠陥散乱)
- `rcwa_solver.py` — RCWA EUV マスク回折解析

**内定戦略**:
- FDTD/RCWAの自作実装が最大の差別化
- GNN欠陥検出 → EUVマスク欠陥検出への応用を語れる
- 中途比率72%で経験者歓迎

## 自分の強みと戦略マッピング

### 現在のスキルセット
- FEM/構造解析 (Abaqus, 自作ソルバ)
- Python, PyTorch, GNN (Graph Neural Network)
- 複合材料 (CFRP) の欠陥検出
- 数値シミュレーション全般

| **ディスコ** | ★★★★★ | Abaqus/FEM経験が完全に直結 |

| **ASML Japan** | ★★★☆☆ | 光学+構造の両方 |

| **アドバンテスト** | ★★☆☆☆ | 電磁界解析は現研究から距離 |
