# 30° セクタ 360° 展開時の境界条件・応力整合性検証

30° バレルセクタを 12 回複製して 360° 円筒に展開したとき、境界条件と応力場の連続性が満たされるか検証する。

---

## 1. 境界条件の設定（generate_gw_fairing.py）

### 1.1 θ = 0° 面（ZSYMM）

| 拘束 | 値 | 物理的意味 |
|------|-----|------------|
| u3 | 0 | 法線方向変位ゼロ（Z 軸に垂直な面） |
| ur1, ur2 | 0 | 回転ゼロ |

- **Shell**: エッジ・フェースに u3=0, ur1=0, ur2=0
- **Solid**: フェースに u3=0 のみ（回転 DOF なし）

### 1.2 θ = 30° 面（局所 CSYS）

- 法線方向が θ 増加方向の局所直交座標系を定義
- **Shell**: u3'=0, ur1'=0, ur2'=0（局所法線方向変位ゼロ）
- **Solid**: u3'=0

---

## 2. 360° 展開の理論的妥当性

### 2.1 対称境界の意味

対称境界条件は「この面が構造の対称面である」ことを仮定する。

- θ=0 と θ=30° は、30° セクタの両端の**対称面**
- 円筒の周期構造では、θ=0 と θ=30° は**同一の物理的境界**（隣接セクタとの接続面）
- 対称面では **σ_n = 0**（法線応力ゼロ）、**u_n = 0**（法線変位ゼロ）

### 2.2 12 回複製時の接続

```
セクタ 0: θ ∈ [0°, 30°]
セクタ 1: θ ∈ [30°, 60°]  ← セクタ 0 の θ=30° 面と接続
セクタ 2: θ ∈ [60°, 90°]
...
セクタ 11: θ ∈ [330°, 360°]  ← θ=360° = θ=0° と接続
```

- セクタ n の θ=30° 面 = セクタ n+1 の θ=0° 面（同じ物理境界）
- 対称 BC により、境界での法線変位・法線応力はゼロ
- 接線方向の変位・応力は、隣接セクタで**連続**（円筒の周期性）

### 2.3 応力の適合性

| 量 | 境界での条件 | 360° 展開時の整合性 |
|----|--------------|---------------------|
| 法線変位 u_n | u_n = 0 | ✅ 対称 BC で保証 |
| 法線応力 σ_n | σ_n = 0 | ✅ 対称面で自動的に成立 |
| 接線変位 u_t | 連続 | ✅ 周期構造で自然に成立 |
| 接線応力 σ_t | 連続 | ✅ 材料・幾何の連続性から |

**結論**: 対称境界条件は、360° 円筒の 30° セクタを正しくモデル化している。12 回複製しても境界での変位・応力は連続する。

---

## 3. 検証方法

### 3.1 ODB 境界ノードの確認（オプション）

境界（θ=0, θ=30°）上のノードで以下を確認:

1. **変位**: u_n ≈ 0（数値誤差範囲内）
2. **応力**: 境界両側で σ_t が連続

Abaqus の Field Output で境界フェースの S, U を抽出し、対称性をチェック可能。

### 3.2 センサ配置との関係

- センサは θ=15° 付近（セクタ中心）にクロス配置
- 境界（θ=0, 30°）にはセンサなし
- ガイド波は境界で反射するが、対称 BC により**鏡面反射**として正しく扱われる

### 3.3 開口部の影響

- HVAC 開口部は θ=20°, z=2500 mm
- 30° セクタ内に含まれる
- 360° 展開時、開口部は 1 箇所のみ（他セクタには存在しない）
- → 開口部は局所的な幾何変更であり、境界の周期icity には影響しない

---

## 4. まとめ

| 項目 | 判定 |
|------|------|
| 境界条件（θ=0, 30°） | ✅ ZSYMM / 局所 CSYS で適切 |
| 360° 展開時の変位連続性 | ✅ 対称 BC により保証 |
| 360° 展開時の応力連続性 | ✅ 周期構造で成立 |
| 開口部の影響 | ✅ 境界から離れており問題なし |

**30° セクタの対称境界条件は、360° 円筒の正しい部分解を与える。展開時の境界・応力の整合性は満たされている。**
